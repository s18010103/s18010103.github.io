# 2.字符串「str」

## 2.1 代码示例

```python
string = " Hello LCY"
t = type(string)
print(string)
print("string type is:>>>", t)
print("直接检测数椐类型, 并输出:>>>", type(string))

#---output---
 Hello LCY
string type is:>>> <class 'str'>
直接检测数椐类型, 并输出:>>> <class 'str'>
```

## 2.2字符串三大特性

1.有序性

1. 「从左到右，下标是从 0 开始」；

2. 「从右到左，小标是从 -1 开始」；

3.  引号里面出现的，都算一个下标 ；

![1](./02-初始数据类型.assets/Image20231004202643.jpg)

![2](./02-初始数据类型.assets/Image20231004203630-1700652258209-3.jpg)
